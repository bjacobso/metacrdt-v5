# MetaCRDT Merge Demo

This static Vite app is the metacrdt.com branch-and-merge proof: two browser replicas assert worker facts independently, cut the cable, exchange §8 version-vector deltas, and converge by recomputing the §5 fold over the §4 G-Set log. The screen maps directly to the protocol: each replica pane shows §3 events and the folded current entity view, the cable reports the L4 Sync delta counts, and the verifier strip recomputes log and projection digests from each actual runtime log on every render. Run it with `pnpm --filter merge-demo dev`, build it with `pnpm --filter merge-demo build`, and run the headless proof with `pnpm --filter merge-demo test`. No server, deployment, or new runtime dependency is used; persistence is the existing `@metacrdt/local` IndexedDB adapter and the headless conformance scenario uses deterministic `@metacrdt/runtime` memory replicas with injected clocks.
